# DATA_ACCESS
Todo el contenido de la asignatura de acceso a datos , estará aquí estructurada. 
## Usabilidad

Dentro del mismo proyecto src , en la siguiente ruta (AccesoADatosTemario/src/main/java/mainFolder) . Tendrémos las carpetas de forma individual por cada temario, también a parte de esto también habrán subcarpetas para lo dado en clase y las prácticas. 

El objetivo es simple, unificar todo el contenido del curso en 1 unico repositorio descargable y accesible.
